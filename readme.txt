Este repositório foi escrito em 2014 durante meu primeiro contato com o TDD.

Em 2019, reescrevi os códigos do zero no seguinte repositório:
https://github.com/alangialorenzo/LivroTDDcomDotNET-2019
Recomendo o repositório recente em detrimento deste antigo.

Em qual situação este repositório antigo é útil?
Versões antigas do Visual Studio não oferecem teste de unidade integrado à IDE para o NUnit.
Caso você precise realizar testes de unidade ou deseje praticar TDD em tais versões antigas, 
este repositório mostra como fazer isso com o NUnit.

Em versões antigas do Visual Studio você precisa do NUnit 2.7.0:
https://github.com/nunit-legacy/nunitv2/releases/tag/2.7.0

Em versões recentes do Visual Studio o NUnit já vem integrado à IDE.
